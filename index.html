Projet :
Détection et classification des tomates en utilisant YOLOv8

YOLO est un modèle pré-entrainé de computer vision qui consiste à prédire les coordonnées des boîtes englobantes et la classe de chaque objet. YOLO réalise simultanément la détection et la classification des objets en une seule passe à travers un réseau de neurones convolutifs

J'ai téléchargé le jeu des données "riped and unriped tomato dataset" disponible sur kaggle. Je n'avais pas besoin de faire les annotations car elles sont déjà fourni dans le téléchargement. J'ai utilisé YOLO version 8 à partir du module "ultralytics" de python. J'ai entrainé mon modèle sur la base des données que j'ai téléchargé car sans cet entrainement, le modèle prédirait les mauvais objets.
